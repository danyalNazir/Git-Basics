Rewriting History:

Undoing commits

git reset --soft HEAD^
git reset --mixed HEAD^
git reset --hard HEAD^

Reverting commits

git revert 72856ea
git revert HEAD~3..
git revert --no-commit HEAD~3..

Recovering lost commits

git reflog
git reflog show bugfix

Amending the last commit

git commit --amend

Interactive rebasing

git rebase -i HEAD~5
