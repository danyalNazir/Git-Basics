Browsing History:

Viewing the history

git log --stat
git log --patch

Filtering the history

git log -3
git log --author=“Mosh”
git log --before=“2020-08-17”
git log --after=“one week ago”
git log --grep=“GUI”
git log -S“GUI”
git log hash1..hash2
git log file.txt

Viewing a commit

git show HEAD~2
git show HEAD~2:file1.txt

Comparing commits

git diff HEAD~2 HEAD
git diff HEAD~2 HEAD file.txt

Checking out a commit

git checkout dad47ed
git checkout master

Finding contributors

git shortlog

Finding the author of lines

git blame file.txt
